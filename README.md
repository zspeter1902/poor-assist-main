#### 软件架构说明

### 前端技术架构

1. Antd-VUE
2. 页面,按钮级别权限控制。
3. 多个组件封装，调用方便。
4. Antv图表组件。
5. WebPack
6. ES6
7. 多环境打包。
8. VUE路由，过滤器，自定义指令。
9. 代码简洁，符合编码规范。

### 后端技术架构

1. SpringBoot2.x
2. Shiro权限框架
3. Redis6.X最新版
4. MyBatis注解版
5. MySQL6.7
6. 分模块开发，自定义启动脚本，JVM调优
7. 多环境,前后端完全分离。
8. 代码生成器。
9. orika传输对象映射器。

#### 安装教程

#### 后端安装方法

```
1.  mvn clean package
2.  tar -zxvf poor-assist-api.gz (解压tar包)
3.  cd poor-assist-api
5.  sh /sbin/startup.sh dev
```

#### 前端安装方法

```
1.  yarn install (安装node_moudle)
2.  yarn start (启动)
3.  yarn build:pro (构建生产包)
```
